# Saddam Hussain — Personal Portfolio 🌐

A clean, mobile-friendly, single-page personal portfolio.
Built for the **Panaversity AI-101 · Lecture 6** task — *"Markdown In, HTML Out."*

The whole site is a single, self-contained **`index.html`** (all CSS and JavaScript inlined) —
no framework, no build step, no dependencies. The look is styled after the
**[Geeky](https://statichunt.com/themes/nextjs-geeky-blog)** portfolio theme, recreated by hand.

## 🔗 Live Demo

**https://saddamchandio.github.io/my-portfolio/**

## ✨ Features

- **Single self-contained file** — one `index.html`, open it anywhere
- **Geeky theme identity** — teal `#2ba283` accent, Raleway + Merriweather Sans fonts,
  wavy-underline section titles, pill buttons, card layout
- **Dark mode** — toggle that respects your system preference and remembers your choice
- **Fully responsive** — mobile hamburger menu, stacked layouts, retina-optimized photo
- **Nice touches** — scroll-reveal animations, animated hero blob, active-nav highlighting, back-to-top
- **Sections:** Hero · About · Technical Core (skills) · Core Projects · Education · Contact
- **Accessible** — semantic HTML, `aria` labels, and `prefers-reduced-motion` support

## 🛠️ The Markdown → HTML Workflow

Following the workflow from class: write yourself in clean Markdown (see [`prompt.md`](prompt.md)),
then iterate with an AI tool to turn it into a polished HTML page.

**Prompts used (in order):**

1. *"Turn this Markdown into a clean, mobile-friendly, one-page portfolio website in HTML. Use clear
   sections, a nice colour theme, and make it look professional. Give me a single HTML file."*
2. *"Match the visual theme (colours, fonts, buttons, section styling) to the Geeky portfolio template."*
3. *"Add a dark-mode toggle, a mobile hamburger menu, and subtle scroll animations."*
4. *"Grade this page out of 10 for design and readability, then improve it."* — iterated on spacing,
   the hero portrait, skill/project cards, and an education timeline.
5. *"Optimize the photo for the web and make sure it looks sharp on mobile."*
